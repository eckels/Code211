Idea Generation Workshop

Friday @ 8:00PM in Idea Room A

Don’t know what you want to build for Code211? In this workshop you will be brainstorming different ideas for your project that you want to build at the Hackathon. This is going to be an idea generating activity, where people will find problems and design creative solutions to solve them. You will also meet other people who may be interested in your idea/solution, or you even may find other projects that pique your interest that you may want to join. This is intended to be a way to help you find a group if you need one, and also to find problems worth solving using Computer Science.

Software Needed:
  • Nothing!

Personal Website Workshop

Friday @ 8:45PM in Idea Room B

Everyone wants their own piece of real estate on the internet, whether it is yourname.com or the next Google, you can learn the basics on how to make it at this workshop. In this workshop you will be working with two of the most common web languages, HTML and CSS.

Twitter Bot Workshop (Python)

Friday @ 9:30PM in Idea Room A

In this workshop we will be building two easy Twitter bots in Python. The first bot will automatically tweet out the weather for a particular location each day. This bot will explore the basics of a Twitter bot and help you get your feet wet with accessing an API and Python fundamentals. The second bot will scramble an image's pixels and tweet it out. This will build upon our past knowledge by using external libraries.

This workshop is at a beginner level and you are not expected to have any Python experience. The bots are simply a means to an end, a way for you to get started programming. Who knows, after this workshop, maybe you could go on to win Code211. There will be a 10 minute initial set up period. We recommend having the newest version of Python installed at the least along with a text editor.

Software Needed:
  • Python
  • Tweepy (Python library)
  • Pillow (Python library)
  • Access to Twitter

Group Chat Messaging Workshop

Friday @ 10:15AM in Idea Room B

In this workshop we'll be creating a group chat messaging website that will be hosted live at the very end! In order to do this, we will utilize Google's Firebase database to store and send messages, while using their hosting to upload our websites to the internet. At the very end, you will be able to send your website URL to friends and have a conversation. Along with that, you will be introduced to servers and databases.

This workshop is at a beginner level but it helps to have prior Javascript experience. However, if you don't have any programming experience, that's completely okay. Who know, maybe those skills could help you go on to win Code211. There will be a 10 minute initial set up period. We recommend having the newest version of node.js installed at the least along with a text editor.

Software Needed:
  • Node.js
  • Gmail Account
  • Code Editor

Twitter Sentiment Analysis Workshop

Saturday @ 8:00AM in Idea Room A

Learn how to do a basic statistical analysis of anyone’s tweets, even your own, using the easy to learn and use Twitter API, Tweepy, a mathematical Python library, Numpy, and Pandas, a Python data analysis library that allows us to aggregate and organize data how we please. Furthermore, learn how to do a simple sentiment analysis of anyone’s tweets using TextBlob, a natural word processor and manipulator that marks any statement or phrase with a positive, negative, or neutral connotation. Do not be intimidated by the seemingly complex nature of this workshop, it is an almost beginner level workshop where we will provide copyable code for anytime you may get lost or confused.

Software Needed:
  • Python
  • Numpy
  • Pandas
  • TextBlob
  • Tweepy

Java GUI Workshop

Friday @ 12:45PM in Idea Room A

Over the course of this workshop, we will:

  • Understand the different aspects of building a Graphical User Interface (GUI) with JavaFX
  • Make our own GUI
  • Learn necessary Java syntax
  • Explore basic Java algorithms and built-in functions

By the end of the workshop, you will end up with more than just a vague understanding of the concepts and features of making a GUI in Java. Not only will you have the practical knowledge necessary to have endless possibilities to make your app, you will walk out of the session with a rough version of a simple application that can easily be expanded upon to help you on your journey to your final product.

Software Needed:
  • Netbeans IDE
  • Scene Builder
